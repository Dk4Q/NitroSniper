## How to:
**You will need:**
- [Node.js](https://nodejs.org/en/ "Node.js")
- [Git](https://git-scm.com/)
- [Yarn](https://yarnpkg.com/)

**Open a terminal and type:**
```
$ git clone https://github.com/migueltsantos/NitroSniper.git
$ cd NitroSniper
$ yarn install
$ node .
```
